## 🎮 게임 종류

### 🖱️ 클릭 배틀
* 일정 시간 안에 최대한 클릭 많이 하기
* 시간 설정 가능
* **(팀전 전용) 이어달리기 모드:** 팀 당 한 명만 클릭 가능하며, 우클릭 시 클릭할 사람 교대 가능

### 🍎 사과 배틀
* 땅따먹기 방식의 사과 배틀
* 드래그로 사과를 선택해서 합이 10이 되면 해당 영역 가져오기 가능

### 🎨 그림 맞히기
* 한 명이 제시어를 보고 그림을 그리면 나머지가 맞히기
* 제한 시간 안에 정답을 맞히면 출제자와 정답자가 점수 획득
* 시간 및 라운드 설정 가능

### ❓ 퀴즈 배틀
* 사진을 보고 답을 맞히기
* 마추기 아이오의 문제를 크롤링하여 가져오기 가능

### 🔢 넘버 러시
* 라운드마다 흩어진 숫자 공을 1부터 순서대로 가장 빠르게 클릭하기
* 제한 시간 안에 더 많이 맞히거나 먼저 끝내면 라운드 승리, 총점으로 최종 승자 결정
* 라운드가 올라갈수록 숫자가 늘어나 난이도 상승

### 🕵️ 라이어 게임
* 한 명은 제시어를 모르는 라이어, 나머지는 같은 제시어를 받음
* 돌아가며 힌트를 말하고, 투표로 라이어를 찾으면 시민 승리
* 제시어 카테고리 및 시간 설정 가능

### 🧠 기억력 게임
* 짧은 시간 동안 공개되는 카드/패턴을 기억
* 점점 복잡해지는 구조 속에서 정확히 맞히면 점수 획득

### ⌨️ 타이핑 레이싱
* 랜덤 문장을 빠르고 정확하게 타이핑해 결승선에 먼저 도착하는 레이스
* 진행도·순위 실시간 갱신, 아이템 상자 획득/공격·방어 효과로 역전 가능

### ⭕ 틱택토
* 3x3 보드에서 번갈아 돌을 놓기
* 가로/세로/대각선 1줄을 먼저 만들면 승리
* 2인 전용
